### Các loại hàm
1. Declaration function
```js
function functionName(){
    ...
}
```
2. Expression function
```js
var myFuntionc = functionName(){
    ...
}
```
Các `callback` như `setTimeout`, `setInteval`,.. cũng là các expression function.

3. Arrow function
```js
function functionName = (arguments) => {
    ...
}
```
